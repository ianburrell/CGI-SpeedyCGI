SpeedyCGI Graph Notes

Creating file_struct.gif:

  • Edit file_struct.crd using the Toolkit for Conceptual Modeling TCRD editor
  • Dump the TCRD window with xwd
  • Crop and convert to gif using gimp
    • Do Tools->Crop to get a general crop
    • Image->Transforms->Autocrop
    • Image->Indexed, choose black/white
    • File->Save As "file_struct.gif"
